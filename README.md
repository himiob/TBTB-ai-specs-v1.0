# TBTB-ai-specs-v1.0
Repositorio oficial de estándares corporativos para desarrollo asistido por IA en TBTB Global. Incluye arquitectura base, guías de backend/frontend, lineamientos de seguridad, modelos, API templates, control de riesgos y reglas para herramientas IA como Cursor/Claude.

TBTB AI-SPECS v1.0
Este repositorio contiene la versión corporativa oficial de los estándares y lineamientos técnicos para el desarrollo asistido por Inteligencia Artificial en TBTB Global.

Objetivo
Proveer una estructura unificada y normativa que garantice:

Consistencia técnica entre proyectos
Arquitectura alineada a estándares de TBTB
Seguridad, privacidad y compliance
Integración efectiva con herramientas IA como Cursor
Control de calidad y mejores prácticas
Uso
Todo nuevo proyecto debe clonar este repositorio dentro de su carpeta raíz como:

/ai-specs
No modifique estos archivos sin aprobación formal del área de Tecnología & Data.

Contenido
Estándares backend, frontend y documentación
Lineamientos de seguridad
Arquitectura base
Especificación de API (OpenAPI)
Modelo de datos
Riesgos técnicos y controles
Guía de desarrollo
Plan de refactorización
Plantillas de prompts para IA
Versionado
Versión actual: v1.0
